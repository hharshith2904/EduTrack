## Student Feedback Management Application
This application facilitates the collection, management, and analysis of student feedback, empowering educators to improve their teaching and learning environment. 
### Features
- User-Friendly Interface: Intuitive design for easy navigation and feedback submission.
- Secure Data Storage: Protected data with user authentication and authorization.
- Flexible Feedback Options: Customizable templates for different feedback types (e.g., course, instructor, specific assignments).
- Data Analysis and Reporting: Tools to summarize and visualize feedback data for informed decision-making.
- Administrator Control Panel: Manage users, feedback templates, and system settings.
- Anonymous Feedback: Option for students to submit feedback anonymously.
### This app includes:
1. Feedback Entry - Collects student feedback
2. Score calculator - Computes the average score
3. Feedback summary - Summarises all the feedbacks into a single feedback file.
4. Report generator - Generates reports for the feedbacks
5. Search feedback - Allows you to search a feedback based on student name.
6. Count feedback - Counts the number of feedbacks
